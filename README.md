# FlutterChat

A real-time messaging app built with Flutter and Firebase.

## Features
- 🔐 User authentication (sign up & login) with Firebase Auth
- 🖼️ Profile picture upload during sign up (camera capture via image_picker)
- 💬 Real-time messaging with Cloud Firestore
- ☁️ Image storage via Firebase Storage
- 🚪 Sign out functionality

## Tech Stack
- **Flutter** — cross-platform UI framework
- **Firebase Authentication** — user sign up/login
- **Cloud Firestore** — real-time database for chat messages
- **Firebase Storage** — profile image storage
